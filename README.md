To run the application locally : 
1) cd into each and every microservice (ms_booking, ms_availability, ms_profiles, ms_service) and run :
2) ./mvnw package && java -jar target/ms_<microservice>-0.0.1-SNAPSHOT.jar
3) cd into FrontEnd
4) run "npm install"
5) run "npm start"
